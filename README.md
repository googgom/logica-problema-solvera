# kolok-neuro-symbolic-logical-problem-solver
Коллоквиум по МЛиТА:  "Нейро-символический решатель логических задач" Архитектура из трех модулей: Модуль 1: LLM-формализатор (Перевод с русского на язык логики) Модуль 2: Движок резолюций (Строгое доказательство) Модуль 3: LLM-объяснятор (Перевод с языка логики на русский)

# Запуск цепочки
python main.py
